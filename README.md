# メイヘン (Meihen)

A modern Japanese landing page designed for property inheritance consultation and document request services.

This project focuses on a clean user experience, responsive layouts, accessibility, and high-performance frontend architecture inspired by contemporary Japanese web design.

---

## ✨ Features

- Modern Japanese UI/UX
- Fully Responsive Design
- Smooth Animations
- Mobile-first Layout
- Reusable React Components
- Interactive Form Validation
- Postal Code Auto Address Lookup (Mock)
- Optimized SVG Assets
- Fast Loading with Vite
- TypeScript Support

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React
- SVG Assets

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/meihen-japan-lp.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
